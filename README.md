CSS Game
==============

Playground for css! write css to match mocks.

To run
====
1. Install [grunt](http://gruntjs.com/getting-started)
2. Install dependencies
    
    ```
    > npm install
    ```
3. Run the project
  
   ```
   > grunt
   ```
4. Open ```localhost:8080``` in modern browser (HTML5 canvas support required)
